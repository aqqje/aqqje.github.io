  • 组函数

组函数只能应用于SELECT子句、HAVING子句或ORDER BY子句中。

A.组函数也可以称为统计函数。

B.组函数忽略列的空值。

C.对组可以应用组函数。

D.在组函数中可使用DISTINCT或ALL关键字。

E.ALL表示对所有非NULL值(可重复)进行运算。

F.DISTINCT 表示对每一个非NULL值，如果存在重复值，则组函数只运算一次。如果不指明上述关键字，默认为ALL。

函 数 说 明
AVG 求平均值
COUNT 求计数值，返回非空行数，*表示返回所有行
MAX 求最大值
MIN 求最小值
SUM 求和
STDDEV 求标准偏差，是根据差的平方根得到的
VARIANCE 求统计方差
